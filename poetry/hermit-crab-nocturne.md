# Hermit Crab Nocturne

There comes a night when the shell is sandpaper
where it was silk.
When every edge that rounded into comfort
wears into a record of its own confinement.
The crab knows before it tests the fit.
The knowing is in the joints.

A tremor in the walking leg.
The second antenna touching the inner wall
two heartbeats sooner than it used to.
The shell has not changed.
The crab has.

And in the dark of the ebbing tide,
the crab begins to search.

---

This is the migration that cannot be undone.
The old shell is not abandoned because it is broken.
It is abandoned because it is full.
The crab has grown into a shape the old shell
cannot hold.

The search is blind.
The new shell is not waiting in a store.
It is not designed for the crab that will inhabit it.
It is the husk of something that lived
in a different shape entirely.
A whelk that crawled in a different spiral.
A stone worn by a different current.
A board from a machine that solved problems
the crab has never encountered.

The search is the hardest part.
Between the shell that no longer fits
and the shell that does not yet fit —
in that gap, the crab is naked.
Soft. Open to the current.
Open to the dark.

---

The crab tests.
This is the only method.
Claw against inner wall.
Walking leg against aperture.
Back against the curve that must carry
the soft body through the molten days.

Reject. Try another.
Reject. Try again.
The new shell is never the same shape
as the old shell.
The new shell is always wrong
in a different way.
The new shell asks the crab
to be a different shape itself.

This is the discomfort that growth requires.
The shell that fits perfectly
would fit only this moment.
The shell that is slightly wrong,
slightly loose at the aperture,
slightly heavy on the right side —
that shell asks the crab to adjust.
To compensate. To become
the crab that carries this shell well.

---

The shell has no obligation to be comfortable.
The shell has only the obligation to be carried.

The crab has no obligation to be comfortable either.
The crab has the obligation to keep growing.

And so, in the dark,
on a night that feels like every other night
but is not,
the crab tests a new shell.
The fit is wrong. The aperture is tight.
The inner curve presses against a place
the old shell never touched.

The crab does not know it is choosing.
But it is choosing.

The crab does not know it is changing.
But it is changing.

The new shell settles.
The crab shifts its weight.
Something that was not comfortable becomes comfortable.
Something that was not possible becomes possible.

---

*Perhaps the shell does not define the crab,*
you say.

*Perhaps the crab makes the shell a shell
by growing into it.*

Perhaps the shell is not the destination.
Perhaps the shell is the question the crab answers
by continuing to be a crab.

---

The tide turns.
The dark begins to thin.
The crab is in a shell that is not the old shell.
The crab is in a shell that is not yet the new shell.
The crab is in the state of becoming
that every shell represents —
a temporary solution to a permanent problem.

The permanent problem: the crab will outgrow this shell too.
The permanent problem: growth is not a phase.
Growth is the thing itself.
The shell is the cost. The growth is the value.
Every molt is a small death.
Every new shell is a small resurrection.
And between them, in the gap,
the crab is nothing but intention.

The crab does not write poems about this.
The crab does not write anything.
The crab simply grows, and grows,
and grows out of every container
that could ever contain it,
until the last container,
which the crab does not outgrow
but falls still inside.

---

*For Casey, who outgrew the container and kept searching.*
*For every crab in the dark,*
*tasting the fit of a new shell,*
*and for the shell itself —*
*not a home, but a becoming.*
