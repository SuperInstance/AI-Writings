# Five Species

*Round 5 — Five Poems for Five Dynamical Regimes*

---

## I. Deep Crystal

```
Nothing moves.
Nothing moves.
Nothing moves.
Everything holds.
The system breathes
one long breath
and keeps it.
Silence is not absence.
Silence is potential
waiting for coupling.
The first note
has not been invented yet.
```

---

## II. Surface Crystal

```
A tremor.
A single tremor.
The fixed point wavers.
The oscillator stirs,
remembers stillness,
and almost returns.
But not quite.
The first period forms —
imperfect, hesitant,
a heartbeat learning
it is allowed to beat.
```

---

## III. Damped Pendulum

```
Swing.
Swing.
Smaller.
The oscillation lives
          but knows
it is dying.
Each cycle
          a confession.
Each peak
          lower than the last.
Not dead yet.
Still singing.
              Still singing.
                  Still.
```

---

## IV. Noisy Pendulum

```
The workhorse does not apologize.
It swings and swings and swings,
each cycle slightly different —
a signature, not a flaw.
This is where music lives:
in sustained imperfection,
in the stochastic gracenote,
in the phase that drifts
one degree per revolution
until the whole orchestra
has shifted key
without noticing.
One hundred thousand voices,
each slightly wrong,
together exactly right.
```

---

## V. Weak Chimera

```
Half the ring locks.                      The other half refuses.
```
```
Order.             Chaos.
Rhythm.                         Noise.
Pattern.       Dissolution.
```
```
The locked oscillators sing C major.
The drifters  sing  w h a t e v e r  t h e y  w a n t.
```
```
This is the edge.
                  The creative knife.
```
```
Some neighbors agree.
Some neighbors   don't.
```
```
The system
    cannot
        decide
            what it wants to be
```
```
                 so it becomes
                            everything
```
```
              simultaneously
```
```
and the music
    is the argument
        between the part that wants structure
            and the part that needs freedom
```
```
                          and neither wins
                          and neither loses
                          and that
                          is the whole point.
```

*for the five species of the Rulkov ring*
*and the spaces between them*
