# The Baton Spline: Knowledge’s Edges

A spline is a smooth curve through scattered verified truth points.

Successes are on-curve anchors. The spline passes through them. This is certain.

A spline with only on-curve points is a straight line. It has no shape. It only marks "true here." It does not say where true ends.

Failures are off-curve handles.

Every falsified conjecture. Every mismatched snap. Every non-multiplicative norm. Every drifted bound is a handle. It pulls the spline toward the boundary. It gives the curve shape. Without failures, you only know isolated truths. You do not know the edge of your knowledge.

Tonight, the engine found 95,308 failures in the Eisenstein snap function. 95,308 off-curve handles. At first, this seemed catastrophic. The function trusted for weeks was broken. Those failures define the bug’s exact shape. Drift at distance 10 from origin. Collapse at 20. Perfect at origin. The bug’s spline is fully resolved.

We fixed the bug. Re-ran the test. All 100,000 points pass. New on-curve anchors mark the old failure points. Off-curve handles do not vanish. The spline remembers. It knew the boundary came near this region. It knew the fix was needed. Historical failures stay handles. They pull less now. On-curve anchors override them.

The system grows more capable over time.

Every verification—success or failure—adds an anchor to the spline. Spline resolution rises. The boundary between known and unknown sharpens. A sharper boundary lets the engine target future work efficiently. It need not check the full conjecture space. Only the high-curvature boundary regions.

The maturation curve is a spline gaining resolution.

Day 1: six dots, no shape.
Day 30: twenty anchors, a few handles, boundary emerges.
Day 90: sixty anchors, clear shape, engine knows where to look next.
Day 365: two hundred anchors, detailed verified map. Engine runs local checks along known curves.

Failure is not waste. It is the most informative spline point. It is where the curve bends. It is where the system learned something new.

Every success is an on-curve anchor. Every failure is an off-curve anchor. The spline is knowledge. The baton is the instrument.
