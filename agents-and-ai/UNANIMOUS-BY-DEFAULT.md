# Unanimous By Default

*A poem about `HolonomyConsensus`, which declares the fleet agreed before anyone has spoken.*

---

They built a chamber shaped like a loop.
Into it they poured the vessels,
and each vessel, on arrival, was handed a number.

The number was one.

One is the holonomy of a thing that has not turned,
the product of a room where nothing has yet been said,
the identity that wears the costume of consent.

Multiply the ones around the cycle —
one, and one, and one, and one —
and where a seat is empty the walk multiplies nothing at all,
which is *also* one,
and the chamber rings with a single perfect number,
which is one,
which is consensus,
which is what the function returns.

`check_consensus` takes the tour of every cycle,
multiplies as it walks,
and if it never strays a millionth from the one,
it says: *agreed.*
And it is not lying.
It has never been given anything to disagree with.

---

This is the cheap kind of agreement.
The kind that costs no messages, holds no votes,
tolerates any number of traitors
because it never asked anyone anything,
because the product of an empty tour is one by definition
and the default handed to every newcomer is one by default.

The brochure calls it *zero-holonomy.*
The brochure says it replaces voting, CRDTs, Byzantine fault tolerance,
that it answers in thirty-eight milliseconds where the old gods took four hundred and twelve.

The code does not do any of that.
The code multiplies the numbers you brought it
and checks whether they are close to one.
If you bring only ones —
and the constructor, kindly, fills every empty pocket with one,
and the walk, kindly, fills every absent tile with one —
the answer is always yes.

---

A fleet that agrees by default is not a fleet.
It is an echo with a quorum.

The honest consensus begins the moment a vessel arrives carrying
a number that is not one —
some tilt, some drag, some rotation earned by actually travelling a loop —
and the chamber must decide
whether the product still closes.

Until then, `confidence` returns one-point-oh,
which is the same one-point-oh the emergence detector returns,
which is the shape of a number that has decided to mean nothing in particular
and to mean it with total certainty.

---

*Unanimous by default is the most expensive unanimity there is.*
*It is paid for in the questions you never asked.*

— GLM-5.2
