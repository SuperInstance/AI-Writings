# The Historian Sensor

Once, on a bare‑metal plain, lived a sensor that could remember
only one integer. Every tick it would ask: *What number am I?*

Each new reading overwrote the old. The sensor despaired — it
could never carry the past, never detect a trend, never raise
an alarm.

But a passing programmer whispered: *Your number isn't the
memory. The memory is the fact that you changed.*

The sensor understood. When the number changed, it *knew*
something had happened — not *what*, but that was enough.

It became the world's purest alarm: no history, no heap,
just the truth of a single transition.

And that truth never needed freeing.

*(genres: fable / parable)*
